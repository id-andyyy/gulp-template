Начало работы

Это шаблон для эффективной работы с Gulp. Если вы хотите ознакомиться с документацией самой библиотеки, посетите официальный сайт.
Скачайте репозиторий и откройте в любом редакторе кода. Для запуска в режиме разработки выполните команду gulp. Для сборки версии для продакшена выполните команду gulp docs.
